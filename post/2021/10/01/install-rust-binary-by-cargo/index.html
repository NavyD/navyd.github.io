使用cargo安装rust二进制文件

2021-10-01 约 249 字 预计阅读 1 分钟 次阅读

文章目录

    • 方法
      • install
      • 检查更新

cargo install管理cargo本地的binary类型文件：bin, example targets

方法

install

1

cargo install evcxr

检查更新

cargo当前未提供直接的命令对已安装的bin检查版本更新，但是提供了一个cargo search命令，可以列出bin的最新版本，与install命令组合可以查看是否可以更新了

1
2
3
4
5
6

$ cargo search --limit 1 evcxr
evcxr = "0.11.0"    # An Evaluation Context for Rust
... and 11 crates more (use --limit N to see more)

$ cargo install --list | grep evcxr
evcxr v0.10.0:

另外，cargo可以通过安装扩展命令 cargo-update更新

1
2
3
4
5
6
7
8

$ cargo install cargo-update
#...

$ cargo install-update -l evcxr
    Updating registry 'https://rsproxy.cn/crates.io-index'

Package  Installed  Latest   Needs update
evcxr    v0.10.0    v0.11.0  Yes
