Powerline in WSL2

2021-10-17 约 303 字 预计阅读 1 分钟 次阅读

文章目录

    • 配置使用

【注意】最后更新于 October 17, 2021，文中内容可能已过时，请谨慎使用。

在wsl2中使用zsh主题出现乱码时需要安装powerline字体才能显示，windows terminal使用的Cascadia Code字体也支持powerline版本 Cascadia Code，常用的还有 powerline/fonts

对于Cascadia Code字体下载 microsoft/cascadia-code/releases解压安装即可：

对于powerline字体文件可以使用git clone https://github.com/powerline/fonts.git --depth=1下载后在管理员权限下执行安装 install.ps1

配置使用

  • vscode：在settings.json中添加

    1
2
3

    {
  "terminal.integrated.fontFamily": "Cascadia Mono PL"
}

  • windows terminal：在settings.json中添加

    1
2
3
4
5
6
7

    {
	"profiles": {
        "defaults": {
            "fontFace": "Cascadia Code PL"
        },
    }
}
