zsh配置

使用zsh包管理器 antigen 比ohmyzsh更方便

安装

1

curl -L git.io/antigen > antigen.zsh

或者Installation

简单配置在$HOME/.zshrc中

source /path-to-antigen/antigen.zsh

# Load the oh-my-zsh's library.
antigen use oh-my-zsh

# Bundles from the default repo (robbyrussell's oh-my-zsh).
antigen bundle git
antigen bundle heroku
antigen bundle pip
antigen bundle lein
antigen bundle command-not-found

# Syntax highlighting bundle.
antigen bundle zsh-users/zsh-syntax-highlighting

# Load the theme.
antigen theme robbyrussell

# Tell Antigen that you're done.
antigen apply

home/end响应修复

写入shell环境：

1
2

bindkey  "^[[F"   end-of-line

fortune

安装fortune和cowsay后在$HOME/.zshrc最后加入：

1
2

# 显示随机名言
fortune | cowsay -f "$(find /usr/share/cowsay/cows/ | shuf -n 1)"

rand-quote plugin

zsh 内置插件rand-quote：Displays a random quote taken from quotationspage.com。在$HOME/.zshrc中加入

1

antigen bundle rand-quote

由于quote要从网络中加载，速度有点慢，影响shell加载速度，可放到后台运行，在$HOME/.zshrc最后加入：

quote | cowsay &
